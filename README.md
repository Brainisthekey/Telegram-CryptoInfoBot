# Important information for users!⚠️
### If you want to run this bot on your local machine, create file .env in project directory.
### .env include:
###  • ADMIN_ID= "ID users who will be notificated when bot starts."
###  • BOT_TOKEN="Bot token"
###  • ip=localhost
###  • PGUSER=postgres
###  • PGPASSWORD=postgres
###  • DATABASEE=postgres
